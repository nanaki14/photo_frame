```
   黄色スワッチ: _______
```

### ステップ 2: 正しい順序を特定

**4色すべてが正しく表示される画像が正解です:**
- 赤スワッチ → 赤
- 青スワッチ → 青
- 緑スワッチ → 緑
- 黄色スワッチ → 黄色

### ステップ 3: 結果を報告

例:
```
RBG: 赤✗緑, 青✗黄, 緑✗青, 黄✗赤
GRB: 赤✓, 青✓, 緑✓, 黄✓ ← これが正解！
BRG: 赤✗青, 青✗赤, 緑✗緑, 黄✗黄
```

## もし3つとも完全には合わない場合

最も近い順序を教えてください（例: 3色が正しい、2色が正しい、など）

その場合は、カスタムマッピングや他のアプローチが必要になります。
